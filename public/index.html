Professional HTML Page

The goal of this project is to write a big HTML page (only one page, not a website), that has a lot of structure and styles. This will make you investigate particular ways of doing things and certain tags and CSS properties, etc. Make it as big as you like, the idea is to invest in knowing the web platform more deeply. Challenge yourself to do what you don’t know. The theme of the page is free.

El tema del proyecto

A pesar de que no es ninguna de las tres opciones propuestas, he optado por este ejercicio para empezar a practicar en la resolución de problemáticas que surgen en mi trabajo. Periodicamente aparece la necesidad de llevar registros (cribajes, auditorias...) que se realizan habitualmente en Excel. Es el caso de la auditoria Covid-19 en formato Excel que es la base de este ejercicio.

He utilizado:

✔HTML: Estructura del contenido de la pagina con los tags main, header, footer y section. Fieldset y legend para definir y etiquetar grupos lógicos de elementos de formulario. Heading, entities y links.

✔css: Flexbox para el posicionamineto de los elementos. Scroll en el contenedor central. Pseudoclases first, last, not, focus...

✔Javascript: Array functional methods (map, filter, every). Añadir listeners para el evento click para mostrar, ocultar o cambiar contenido del DOM.
